# The One 🎮

**Roguelike Stationary Hero Defense**

Stationary hero. Endless swarms. Gacha-style loot rolls. Four unique heroes.

## Status
- **Phase:** Pre-production / GDD
- **Engine:** Unity (2D, URP)
- **Dev approach:** Claude Code as primary coder, Gary as project manager

## Key Files
- `GDD.md` — Full Game Design Document (give this to Claude Code)
- `unity/` — Unity project (TBD, created via Unity Hub)

## Setup
1. Unity Hub installed ✅
2. Install Unity Editor (2022 LTS or 6000 LTS) via Unity Hub
3. Create new 2D URP project in `the-one/unity/`
4. Load GDD.md into Claude Code and start building

## Heroes
- 🏹 Archer — DPS glass cannon
- 🔮 Wizard — AoE powerhouse  
- 🦴 Cannibal — Sustain tank
- 🟢 Slime — Area denial / summoner

## Inspirations
- The Tower (mobile) — stationary hero concept
- Megabonk — RNG gamba dopamine
