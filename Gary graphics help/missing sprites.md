# Missing Sprites for The One

## Current Art Style
- **Perspective**: Top-down view
- **Style**: Clean, simplified vector art with soft shading
- **Colors**: Warm palette - browns, golds, greens, muted tones
- **Shape Language**: Rounded, friendly forms with thick outlines
- **Size**: Characters ~512x512px, items/effects ~128x128px

## Priority 1: Core Heroes

### 1. The Noob (Starting Character)
- Simple peasant/villager character
- Tattered brown tunic/rags
- Wooden stick or slingshot as weapon
- Slightly hunched, unsure posture
- Warm brown/beige colors to match archer style

### 2. Fighter Class
- Sturdy warrior with basic armor
- Leather/chainmail vest, simple helmet
- One-handed sword and small shield
- Broader build than archer
- Earth tones - browns, metallic grays

### 3. Apprentice/Mage Class  
- Young magic user in simple robes
- Pointed wizard hat (smaller than full wizard)
- Wooden staff with small crystal
- Purple/blue robes to match wizard theme
- Slimmer build, scholarly appearance

## Priority 2: Visual Effects

### 4. XP Orbs (3 sizes)
- Glowing blue/cyan crystalline orbs
- Small (5 XP), Medium (10 XP), Large (25 XP)
- Soft glow effect, semi-transparent
- Diamond/gem shape to match current placeholder

### 5. Explosive Arrow Impact
- Circular explosion burst
- Orange/yellow fire colors
- 3-4 frame animation sequence
- Matches style of dust cloud effects

### 6. Mark for Death Indicator
- Red skull or crosshair above enemy
- Semi-transparent overlay effect
- Pulsing/glowing animation
- Clear but not overwhelming

### 7. Rain of Arrows Effect
- Multiple arrows falling from above
- Semi-transparent to show impact area
- Golden/yellow tint for visibility
- Matches archer arrow style

## Priority 3: Boss Enemies

### 8. Goblin Chief (First Boss)
- Larger goblin with crown/headdress
- Ornate staff or dual daggers
- More detailed armor/clothing
- Red/gold accents on green skin
- 2x size of regular goblin

### 9. Skeleton Lord
- Armored skeleton with cape
- Glowing eyes, ornate sword
- Crown or helmet with horns
- Purple/black color accents
- More imposing stance

### 10. Bat Swarm Queen
- Giant bat with crown/tiara
- Spread wings, intimidating pose
- Purple/black coloring
- Glowing red eyes
- 3x size of regular bat

## Priority 4: UI Elements

### 11. Ability Icons (64x64px)
Match item icon style but for abilities:
- Steady Aim: Crosshair/target
- Quick Draw: Arrow with speed lines
- Double Nock: Two arrows
- Explosive Tips: Arrow with flame
- Hunter's Focus: Eye with arrow
- Rain of Arrows: Multiple arrows from clouds
- Marked for Death: Skull with X
- Wind Arrows: Arrow with wind swirls

### 12. Class Selection Portraits
- Larger character portraits (256x256px)
- Show class personality/style
- Consistent framing and backgrounds
- Match warm color palette

## Priority 5: Environmental

### 13. Destructible Props
- Wooden barrels
- Crates/boxes
- Small rocks
- Match ground tile style
- Break into pieces when destroyed

### 14. Arena Boundaries
- Stone walls or wooden fences
- Modular pieces for variety
- Same perspective as characters
- Earthy, muted colors

## Technical Requirements
- PNG format with transparency
- Consistent outline thickness (3-4px)
- Soft shadows beneath characters
- All sprites centered in canvas
- Power-of-2 dimensions when possible

## Reference
Existing sprites use a distinctive style:
- Thick, dark outlines
- Two-tone shading (base + shadow)
- Slightly desaturated colors
- Friendly, approachable design
- Clear silhouettes for readability
